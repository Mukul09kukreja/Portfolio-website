Prerequisites

A modern web browser (Chrome, Firefox, Safari, Edge)
Text editor (VS Code, Sublime Text, etc.) for editing

Installation & Setup

Clone the repository

bashgit clone https://github.com/Mukul09kukreja/Portfolio-website.git

Navigate to the project directory

bashcd Portfolio-website

Open in browser

Simply open portfolio website/index.html in your browser
Or use Live Server for development



Using Live Server (Recommended for Development)
If you're using VS Code:

Install the "Live Server" extension
Right-click on index.html
Select "Open with Live Server"

In index.html, find the projects section and add:

    <h3>Project Title</h3>
    <p>Project description...</p>
    <a href="project-link">View Project</a>
</div>
ğŸ¨ Color Customization
To change the color scheme, update these CSS variables in your stylesheet:
css:root {
  --primary-color: #your-color;

  --text-color: #your-color;
  --background-color: #your-color;
}
